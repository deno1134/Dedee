# ⚡ Denicord - Quick Start Guide

## 🚀 **3-Minute Deployment**

### 1. **GitHub Repository Setup**
```bash
# Fork or clone this repository
git clone https://github.com/yourusername/denicord.git
cd denicord

# Push to your GitHub repository
git remote add origin https://github.com/yourusername/denicord.git
git push -u origin main
```

### 2. **Render.com Deployment**

#### Step 1: Create PostgreSQL Database
1. Go to [Render.com](https://render.com)
2. Click **"New"** → **"PostgreSQL"**
3. Settings:
   - **Name**: `denicord-db`
   - **Database**: `denicord`
   - **User**: `denicord_user`
   - **Region**: `Frankfurt (EU)`
   - **Plan**: `Free`
4. Click **"Create Database"**

#### Step 2: Deploy Backend Server
1. Click **"New"** → **"Web Service"**
2. Connect your GitHub repository
3. Settings:
   - **Branch**: `main`
   - **Root Directory**: `server`
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Plan**: `Free`

#### Step 3: Configure Environment Variables
```env
NODE_ENV=production
DATABASE_URL=[Link to your PostgreSQL database]
JWT_SECRET=[Auto-generated by Render]
JWT_EXPIRES_IN=7d
CORS_ORIGIN=*
MAX_FILE_SIZE=50MB
UPLOAD_DIR=/tmp/uploads
```

#### Step 4: Deploy Frontend
1. Click **"New"** → **"Static Site"**
2. Connect your GitHub repository
3. Settings:
   - **Branch**: `main`
   - **Root Directory**: `desktop`
   - **Build Command**: `npm install && npm run build`
   - **Publish Directory**: `dist`

#### Step 5: Configure Frontend Environment
```env
VITE_API_URL=https://your-backend-url.onrender.com/api/
VITE_WS_URL=wss://your-backend-url.onrender.com
```

### 3. **Database Schema Setup**

```bash
# Wait for database to be ready
# Copy External Database URL from Render dashboard
export DATABASE_URL="postgresql://denicord_user:password@host:5432/denicord"

# Deploy schema
psql $DATABASE_URL -f server/database/schema.sql
```

### 4. **Test Your Deployment**

#### Backend Health Check:
```bash
curl https://your-backend-url.onrender.com/api/health
```

#### Create Test User:
```bash
curl -X POST https://your-backend-url.onrender.com/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "password123"
  }'
```

### 5. **Build Mobile Apps**

#### Android APK:
```bash
cd app
./gradlew assembleRelease
# APK: app/build/outputs/apk/release/app-release.apk
```

#### Upload to GitHub Releases:
1. Go to your GitHub repository
2. Click **"Releases"** → **"Create a new release"**
3. Tag: `v1.0.0`
4. Upload APK file
5. Publish release
